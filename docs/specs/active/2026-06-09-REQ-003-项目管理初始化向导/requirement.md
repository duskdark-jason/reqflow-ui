# 项目管理初始化向导前端需求说明

## 背景

REQ-002 已把项目接入中心、仓库和客户基线维护入口、MCP 索引指引、影响面推荐基础能力落到前端，但仍没有达到最初设想的项目管理菜单体验。管理员在“新增项目”时仍只能维护项目基础字段；前后端仓库、统一客户基线、模块功能点和索引状态仍需要进入项目接入中心或其他菜单补齐。

本需求要把后台“项目管理”改造成真正的项目初始化入口：管理员点击新增项目即可按步骤填写项目名称、前端仓库、后端仓库、客户或地区统一基线分支，并看到模块功能点初始化状态。平台不保存本机目录；本机目录仅作为本地索引器执行时的临时输入或提示，不进入后端保存。

## 目标

- 将项目管理新增/编辑改造成初始化向导，而不是单一项目基础字段弹窗。
- 在项目管理菜单内连续维护项目基础信息、前端仓库、后端仓库、客户基线和初始化确认。
- 在项目列表中展示初始化状态，让管理员能快速识别项目是否具备仓库、客户基线、模块知识和索引批次。
- 在初始化向导完成页提供 MCP 索引下一步指引，但不把 MCP 作为管理员必须理解的唯一入口。
- 保留项目接入中心详情页作为项目后续维护入口，不重复造一个孤立页面。

## 范围

本次包含：

- 改造 `src/views/requirement/project/index.vue` 的新增/编辑入口，使用分步向导维护项目初始化信息。
- 新增或调整初始化 API 封装，调用后端 companion 的项目初始化聚合接口。
- 向导步骤包含：项目信息、代码仓库、客户基线、模块初始化状态、确认保存。
- 仓库步骤默认给出前端仓库和后端仓库两行，可增删其他仓库；只保存团队共享 Git 远端和默认分支。
- 客户基线步骤默认给出主线或首个客户基线，可维护客户名称、客户线编码、统一基线分支和分支策略。
- 项目列表展示初始化状态标签或进度摘要，并提供“继续初始化/接入中心”入口。
- 更新前端 harness 文档，说明项目管理菜单是初始化主入口。

本次不包含：

- 浏览器直接扫描本机仓库或读取本机文件系统。
- 前端保存个人绝对路径。
- 重新实现 MCP 索引器。
- 新增复杂可视化依赖或替换 Element UI。
- 移除已有仓库管理、客户定制线和模块功能点菜单。

## 影响范围

- 接口/API：是，新增项目初始化聚合 API 封装。
- 数据库/SQL：否，前端不直接改数据库；依赖后端 companion。
- 权限/菜单：是，项目管理新增/编辑向导需要组合项目、仓库、客户基线权限。
- 页面/交互：是，项目列表、新增项目、编辑项目、项目接入中心入口都会调整。
- 导出/异步/任务：否。

## 契约与数据口径

- API 文件建议：
  - `src/api/requirement/projectInit.js` 封装 `getProjectInit`、`addProjectInit`、`updateProjectInit`。
- 页面入口：
  - `src/views/requirement/project/index.vue`：项目管理列表、初始化状态、向导入口。
  - `src/views/requirement/project/detail.vue`：项目接入中心详情页，保留作为后续维护和索引查看入口。
- 向导步骤：
  - 步骤一“项目信息”：项目名称、项目编码、负责人、说明、状态。
  - 步骤二“代码仓库”：仓库名称、仓库类型、Git 地址、默认分支、状态；默认包含前端和后端仓库。
  - 步骤三“客户基线”：客户线名称、客户线编码、客户名称、统一基线分支、分支策略。
  - 步骤四“模块初始化”：展示模块数、索引批次、最近索引、下一步 MCP 指引；不在此步骤保存本机目录。
  - 步骤五“确认保存”：汇总项目、仓库、客户基线和待补事项。
- 数据粒度：
  - 列表一行一个项目，并展示该项目初始化状态。
  - 向导一次编辑一个项目的初始化上下文。
  - 仓库一行一个团队共享 Git 仓库。
  - 客户基线一行一个客户或地区基线。

## 验收标准

- AC-UI-001：管理员点击项目管理“新增项目”后看到分步初始化向导，而不是只有项目基础字段的单页弹窗。
- AC-UI-002：向导可维护项目名称、项目编码、前端仓库、后端仓库、客户基线和统一基线分支。
- AC-UI-003：向导和项目列表明确提示平台不保存本机目录，只保存共享 Git 信息和基线分支。
- AC-UI-004：项目列表展示初始化状态，能区分缺仓库、缺客户基线、缺模块知识、已完成基础初始化。
- AC-UI-005：编辑已有项目时可回显项目、仓库和客户基线，并保存修改后仍返回项目列表或进入接入中心。
- AC-UI-006：前端 harness 文档同步更新项目管理菜单、初始化向导和接口调用契约。
