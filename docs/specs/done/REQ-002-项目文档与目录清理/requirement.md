# 项目文档与目录清理需求说明

## 背景

前端 README 仍是若依默认开发片段，无法说明 ReqFlow 当前项目定位。通用文档模板中的 SQL 路径也需要与后端 `docs/db/sql/` 迁移保持一致。

## 范围

- 重写前端 README。
- 同步前端 `docs/` 中的 SQL 路径规则和执行报告模板。
- 配合后端完成项目文档清理记录。

## 不在范围

- 不修改页面、路由、API 封装或构建配置。

## 验收标准

- AC-001：前端 README 不再是若依默认启动说明。
- AC-002：前端文档模板使用 `docs/db/sql/` 或 `docs/db/` 记录数据库相关路径。
- AC-003：文档和 harness 校验通过。
