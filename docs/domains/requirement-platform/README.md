# 需求管理平台前端领域入口

## 业务边界

本领域承接需求管理平台前端能力，包括项目管理、仓库管理、客户定制线、模块功能点、需求提交、执行包查看、MCP 配置和使用统计页面。

## 当前状态

- 需求管理业务页面已创建，覆盖项目管理、项目接入中心、仓库、客户基线、模块功能点、需求、Agent 交接资料和统计页面。
- 项目管理已改造为项目初始化主入口，新增和修改项目时通过单页维护弹窗维护项目基础信息、前后端仓库、分支中文标签、真实分支名和模块初始化状态。
- 前端 API 封装已覆盖 `/requirement/project/init/**`、基础 CRUD、仓库索引、需求、执行包和统计接口。
- 前端不保存个人本机仓库目录；本机目录只属于本地索引器临时读取范围。

## 后续维护

- 新增页面、路由、按钮权限或 API 调用时，同步补充 `docs/ai-harness/modules/` 或 `docs/ai-harness/contracts/`。
- 新增长期有效交互规则时，同步补充 `docs/ai-harness/decisions/`。
